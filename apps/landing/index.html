Game of Thrones REST API

gotapi

Personagens, dragões, espadas e casas dos Sete Reinos, disponíveis via HTTP.

Endpoints Como Rodar
base url http://localhost:5000/api

Endpoints

4 rotas disponíveis

GET ⚔️

/characters

/characters/<id>

Nome, casa, status e títulos de cada personagem.

GET 🐉

/dragons

/dragons/<id>

Nome, cor, dono atual e status de vida de cada dragão.

GET 🗡️

/swords

/swords/<id>

Espadas lendárias com material, dono e origem.

GET 🏰

/houses

/houses/<id>

Casas de Westeros com brasão, palavras e membros.

Instruções de execução

1. Iniciar o backend

cd apps/backend
uv run flask --app src/app run --debug

2. Iniciar o frontend

cd apps/frontend
uv run flet run main.py

3. Abrir a documentação

Acesse http://localhost:5000/apidocs no navegador para ver a documentação Swagger do projeto.
